# 项目文档索引

`docs/` 用来集中存放项目研究资料、系统设计文档、论文写作素材和产品技术资料。

## 目录说明

| 路径 | 内容 |
|------|------|
| `研究资料/` | 项目背景、流程说明、配套图片等研究资料 |
| `系统设计/` | 系统架构、功能设计、需求分析等系统设计文档 |
| `开发计划/` | 过程计划、阶段任务、实现安排等开发记录 |
| `论文写作/` | 论文正文素材、页面实现文段、章节草稿 |
| `产品与技术资料/` | PRD、优化方案、测试清单等产品与技术文档 |
| `导出-Word用/` | 供 Word 打开或导出的 HTML、TXT、PDF 文件 |
| `../backend/docs/` | 数据库设计、ER 说明、后端配套文档 |
