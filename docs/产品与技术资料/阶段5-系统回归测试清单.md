# 阶段5：系统回归测试清单

本文档用于本轮数据库重构、接口对齐、后台与移动端功能调整后的回归测试。测试时优先覆盖“登录、社区绑定、公益服务单、志愿者认证、重点关怀、便民信息、异常预警、数据统计”这些主线功能。

## 1. 测试入口

| 入口 | 地址 | 说明 |
| --- | --- | --- |
| 后端服务 | http://localhost:8080 | Spring Boot 接口服务 |
| 管理端 | http://localhost:7000 | 社区管理员、平台管理员后台 |
| 移动端 | http://localhost:9000 | 居民、志愿者混合使用端 |

## 2. 测试账号

| 账号 | 密码 | 角色 | 重点测试 |
| --- | --- | --- | --- |
| manager | 已脱敏 | 社区管理员 | 审核、公告、便民信息、预警 |
| resident1 | 已脱敏 | 居民 | 发布求助、查看公告、申请加入社区 |
| volunteer1 | 已脱敏 | 志愿者 | 接单、完成服务、查看推荐 |
| pending1 | 已脱敏 | 待审核居民 | 社区加入审核流程 |

## 3. 登录与基础访问

| 测试点 | 操作 | 预期结果 |
| --- | --- | --- |
| 未登录访问后端首页 | 打开 http://localhost:8080 | 返回未登录提示，属于正常安全拦截 |
| 管理端登录 | 使用 manager 登录 | 进入后台首页，侧边栏正常显示 |
| 移动端登录 | 使用 resident1 登录 | 进入移动端首页，用户信息正常 |
| Token 过期处理 | 清除登录态后刷新页面 | 跳回登录页或提示重新登录 |

## 4. 移动端首页

| 测试点 | 操作 | 预期结果 |
| --- | --- | --- |
| 轮播图 | 打开移动端首页 | 显示固定公益主题轮播图，不再报 SQL 错误 |

| 异常兜底 | 后端无轮播数据时刷新 | 自动使用 Picsum 固定图，不影响公告显示 |

## 5. 社区绑定流程

| 测试点 | 操作 | 预期结果 |
| --- | --- | --- |
| 居民申请加入社区 | pending1 提交社区邀请码或申请 | 生成待审核记录 |

| 数据隔离 | 切换不同社区管理员 | 只能查看本社区数据 |
